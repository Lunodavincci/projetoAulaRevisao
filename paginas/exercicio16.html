Tecnologias Web Fundamentais

HTML5

HTML (HyperText Markup Language) é a linguagem de marcação padrão para criar páginas web.

  • Estrutura o conteúdo da página
  • Define a semântica do documento
  • Permite incorporar multimídia
  • Cria formulários interativos

CSS

CSS (Cascading Style Sheets) é usado para estilizar elementos escritos em HTML.

JavaScript

JavaScript é uma linguagem de programação que permite implementar itens complexos em páginas web.
